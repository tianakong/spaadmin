创建Symfony页面

创建一个新页面 - 无论是HTML还是JSON输出——都是一个简单的“两步”操作：

1.创建一个路由：路由是网页的URL（例如/ about）并指向控制器;

2.创建一个控制器：控制器是你编写和建立页面的功能。 你接收传入的请求信息并使用它创建一个Symfony Response对象，该对象可以保存HTML内容，JSON字符串或甚至像图像或PDF这样的二进制文件。

创建一个页面：路由和控制器
注意
在继续之前，请确保您已阅读安装章节并可在浏览器中访问您的Symfony应用程序。

假设你想创建一个页面 - /lucky/number - 生成一个幸运的数字并打印出来。 那么，要先创建一个控制器类并添加方法：

// src/Controller/LuckyController.php
namespace App\Controller;

use Symfony\Component\HttpFoundation\Response;

class LuckyController
{
    public function number()
    {
        $number = mt_rand(0, 100);

        return new Response(
            'Lucky number: '.$number.''
        );
    }
}

现在，你需要将这个控制器函数与一个公共URL(例如/lucky/number)关联起来，以便在用户浏览它时执行number()方法。这个关联是通过在config/routes.yaml文件中创建路径来定义的: 

# config/routes.yaml

# the "app_lucky_number" route name is not important yet
app_lucky_number:
    path: /lucky/number
    controller: App\Controller\LuckyController::number

就是这样!如果您使用Symfony web服务器，请尝试以下操作：

http://localhost:8000/lucky/number
