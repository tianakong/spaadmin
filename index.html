安装和设置Symfony框架

欢迎来到Symfony！开始一个全新的Symfony项目是容易的，你的第一个可正常运作的Symfony程序能在几分钟之内配置完成。

安装Symfony

要创建新的Symfony应用程序，首先确保您使用PHP 7.1或更高版本并安装了Composer。 否则，请先在您的系统上全局安装Composer。 如果您想使用虚拟机（VM），请查看Homestead。

用Composer创建Symfony程序:

composer create-project symfony/website-skeleton my-project

这将创建一个新的my-project目录，下载一些依赖关系，甚至生成你需要开始的基本目录和文件。 换句话说，你的新应用已经准备就绪！

注意
框架针对传统的Web应用程序进行了优化。 如果您正在构建微服务，控制台应用程序或API，请考虑使用更简单的框架项目： 
composer create-project symfony/skeleton my-project
运行Symfony应用程序

在开发时，您应该使用Nginx或Apache等Web服务器（请参阅配置Web服务器以运行Symfony）。 但是对于开发来说，使用Symfony PHP Web服务器更加容易。

首先，进入你的新项目并安装服务器：

cd my-project
composer require server --dev

要启动服务器，请运行：

php bin/console server:run

打开浏览器并导航到http://localhost:8000/。 如果一切正常，您会看到一个欢迎页面。 稍后，当您完成工作时，请通过从终端按Ctrl + C来停止服务器。

检查安全漏洞

Symfony提供了一个名为“Security Checker”（或sec-checker）的实用程序来检查项目的依赖关系是否包含任何已知的安全漏洞。 运行以下命令将其安装到您的应用程序中： 

cd my-project/
composer require sec-checker --dev

从现在开始，只要您在应用程序中安装或更新任何依赖项，该实用程序就会自动运行。 如果依赖包含漏洞，您将看到一条清晰的消息。

Symfony Demo应用程序

Symfony演示应用程序是一个功能齐全的应用程序，显示了开发Symfony应用程序的推荐方式。 它是Symfony新手的一个很好的学习工具，它的代码包含大量的评论和有用的注释。

执行此命令来安装项目： 

composer create-project symfony/symfony-demo
